===Description===

[PHP5 is required]

=== Installation ==

You can either install it automatically from the wordpress admin panel or do it manually:

1. Unzip the archive and put the nepali-unicode-converter folder into your wordpress folder(/wp-content/plugins/).
2. Activate the plugin from the Plugins menu.


==Usage==

1. After activating the plugin you will see a field "Nepali Unicode" in the admin 	   settings menu
2. Click on that and you will be taken into the page where you can convert any text 
   into Nepali unicode and paste them anywhere in the web.
3. It needs internet connection to work.
